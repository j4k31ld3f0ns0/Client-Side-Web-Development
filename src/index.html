What is JavaScript?

JavaScript (not to be confused with a similarly-named but very different Java programming language) is a programming language that is most well-known as the scripting langauge for Web pages, but it also has many other non-browser applications, with notable examples existing like Apache CouchDB, Node.js, and Adobe Acrobat.
