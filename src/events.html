Events

Event-based programming is another fairly straightfoward concept with a lot of utility, especially in a web-development setting, which JavaScript is known for.
Essentially the idea is that there are predefined uses or interactions for a given application, and upon receiving an input, the program will react with an event, so you can visualize it a lot like a bear trap, it doesn't really do anything until you step in it. (Haha)
