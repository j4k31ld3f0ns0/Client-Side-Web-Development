Arrays

Arrays are arragements of data, whether it be objects, pictures, or numbers, where multiple pieces of data that are related are stored in similar locations so they can be quickly grouped together, accessed, and otherwised modified in a straightfoward manner. It is one of the fastest, most basic data structures used in computing, and can be typically built upon to create larger, more complex data structures for a variety of use cases.

Pictured below is a visual example of what an array is, and when it is defined in Javascript using the proper syntax, you can access and modify data by using the indicies of the array (For example, myArray[0] would reference the first item)

Overall, the implementation and usage of them is fairly straightfoward, although you can do lots of fun stuff with them, like create complex data structures using things like Objects.
